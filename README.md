# Yukihana-docs

这是一份适用于 Yukihana 机器人的文档。

## 运行项目

1. 拉取项目代码 `git clone https://github.com/cuit-edu-cn/Yukihana-docs.git`
2. 安装依赖 `pnpm install`
3. 运行开发环境 `pnpm run docs:dev`
